Discord Verification

Enter your Discord User ID to verify access to the download.

How to find your Discord User ID:

  1. Open Discord Settings
  2. Go to "Advanced" under "App Settings"
  3. Enable "Developer Mode"
  4. Right-click on your username in any server
  5. Copy your User ID at the bottom

Enter Discord User ID

Your Discord ID is only used to verify your access role in the Discord server.
